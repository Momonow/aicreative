# Stacked Format

Use a stable vertical split: proof or context media above, presenter below. Campaign imagery,
caption style, and legal text remain project inputs.

## Default Geometry

- Portrait canvas with equal top and bottom panels as a starting point.
- Top panel: approved B-roll, stills, screenshots, or proof.
- Bottom panel: presenter cropped to keep the face prominent.
- Detect face position per presenter; do not reuse one fixed crop across a batch.
- Let the seam and each subject retain breathing room.

Adjust the split only when the reference or source framing requires it.

## B-Roll

- Lead with the strongest hook-relevant asset.
- Order proof first, then context, then texture.
- Cut on narrative beats and maintain natural spacing.
- Vary the opening asset across a batch.
- For stills, use subtle, varied crop motion; avoid repeating one zoom recipe.
- Never substitute generic paperwork for human or diagnosis proof when the spoken beat requires it.

## Captions

- Preview the actual composite and place captions around the seam only when they remain legible
  over both panels.
- Fit the widest rendered card to the mobile viewport.
- Apply any campaign disclaimer as a separate layer.

## Workflow

1. Approve presenter and every proposed top-panel asset.
2. Detect and review the presenter's crop.
3. Assemble one clean pilot at native speed.
4. Preview caption placement.
5. Render clean, captioned, and disclaimer versions separately.
6. Run transition sheets and every-frame QA before batching.
